# Fighter Jet HUD Simulator (戦闘機HUDシミュレーター)

TypeScript と HTML5 Canvas を使用した、軽量でインタラクティブな戦闘機ヘッドアップディスプレイ（HUD）シミュレーターです。機体の姿勢（ロール、ピッチ）、飛行データ（速度、高度、方位）の変化をリアルタイムに計算し、2D Canvas 上にベクトルグラフィックとして描画します。

## 🚀 特徴

- **リアルタイム姿勢レンダリング**: 座標変換（`translate` / `rotate`）を用いた、滑らかで正確なピッチラダーの回転・移動。
- **計器テープのスクロール**: クリッピング機能（`clip`）を使用した、ハミ出しのない速度・高度テープメーターの実装。
- **モダンなビルド環境**: 軽量・高速な開発サーバー「Vite」による TypeScript 環境。

## 🛠️ 必要条件

- **Node.js**: v18 以上推奨
- **npm**: v9 以上推奨

⚙️ 主要ロジックの解説（src/hud.ts）
1. 座標系の分離
ctx.save() と ctx.restore() を使用することで、機体の傾き（ロール・ピッチ）に追従して回転・移動する「中央の姿勢インジケーター領域」と、画面に固定されている「左右の速度・高度メーター領域」の描画ロジックを完全に独立させています。

2. 回転と移動の順序
ピッチラダーの描画では、数式で座標を一点ずつ回転計算するのではなく、Canvasのコンテキスト自体を以下のように変形させて描画を単純化しています。

ctx.translate(centerX, centerY) で中心に移動

ctx.rotate(-roll) で画面全体を回転

回転した軸に沿ってピッチ（上下位置）をループ描画

3. メーターのマスク処理
速度・高度テープは、指定した矩形領域（ctx.rect）で ctx.clip() を実行することで、メーター領域の外側に数字や目盛りがハミ出さないように制御しています。
